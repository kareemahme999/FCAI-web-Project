## 📌 Project Overview

The **Online Library Website** is a web-based application that allows users to browse, search, and borrow books online.
The system supports two types of users: **Admin** and **Regular User**, each with different permissions and functionalities.

This project is developed as part of the Web Technology course.

---

## 🎯 Project Objectives

* Provide a user-friendly interface for browsing and borrowing books.
* Allow administrators to manage books efficiently.
* Implement authentication and role-based access control.
* Dynamically adjust navigation elements based on user roles.

---

## 👥 User Roles

### 🔐 Admin

Admin users have full control over book management.

#### Admin Capabilities:

* Sign up (username, email, password, confirm password, is_admin)
* Login
* Add new books
* View list of available books
* Edit book details
* Delete books

#### Book Attributes:

* Book ID
* Book Name
* Author
* Category
* Description
* Availability Status

---

### 👤 Regular User

Regular users can browse and borrow books.

#### User Capabilities:

* Sign up (username, email, password, confirm password, is_admin)
* Login
* Search books by:

  * Title
  * Author
  * Category
* View available books (Available / Not Available)
* View book details
* Borrow available books
* View list of borrowed books

---

## 🧭 Navigation Bar

* A dynamic navigation bar will be implemented.
* The navigation bar will:

  * Adjust based on whether the user is logged in.
  * Display different options for Admin and Regular Users.
  * Be accessible from all website pages.
